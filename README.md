# nvh-pre-sales-research
NVH/声学售前市场调研技能。输入客户名称和目标产品，自动调研→Markdown报告→HTML报告全流程。
输出：(1) 结构化售前调研Markdown报告 (2) 基于标准模板的HTML网页报告。
触发词：售前调研、客户调研、异音分析、NVH需求、竞品异响、客户投诉分析、产品异响调研、打单准备、客户需求画像、异音检测需求、NVH售前、声学售前。  
  用户输入客户+产品                                                                                                                  
      ↓                                                                                                                              
  第零步：信息收集确认                                                                                                               
      ↓                                                                                                                              
  第一步~第四步：调研执行（客户画像→异音画像→反馈挖掘→需求匹配）                                                                     
      ↓                                                                                                                              
  第五步.1：输出 Markdown 报告                                                                                                       
      ↓                                                                                                                              
  第五步.2：读取 HTML 模板 → 填入数据 → 生成 HTML 报告（必须）                                                                       
      ↓                                                                                                                              
  浏览器确认效果
